AVAudioSession
==============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Controls the shared iOS/tvOS audio session from GDScript.

.. rst-class:: classref-introduction-group

Description
-----------

Use this helper to select one of Apple's predefined ``AVAudioSession`` categories so your Godot project can mix or duck audio according to platform expectations. It is most useful on mobile builds where the engine needs to share the audio device with the system music player or with voice chat as described in ``GameCenterGuide.md``. See Apple's audio session overview at `Apple's AVAudioSession reference <https://developer.apple.com/documentation/avfaudio/avaudiosession>`__.
