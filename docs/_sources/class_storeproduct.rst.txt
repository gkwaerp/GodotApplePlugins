StoreProduct
============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents a product available for purchase in the App Store.

.. rst-class:: classref-introduction-group

Description
-----------

This class contains information about a product, such as its ID, display name, description, and price. It is returned by :ref:`StoreKitManager.request_products()<class_StoreKitManager_method_request_products>`.
