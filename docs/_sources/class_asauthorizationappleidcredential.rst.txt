ASAuthorizationAppleIDCredential
================================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents a credential that results from a successful Apple ID authentication.

.. rst-class:: classref-introduction-group

Description
-----------

This class contains information about the user authenticated via Sign in with Apple. It includes the user identifier, full name, email, and authentication tokens.



The :ref:`identity_token<class_ASAuthorizationAppleIDCredential_property_identity_token>` and :ref:`authorization_code<class_ASAuthorizationAppleIDCredential_property_authorization_code>` properties are provided as :ref:`PackedByteArray<class_PackedByteArray>`\ s, which can be sent to your backend server for verification.



The :ref:`real_user_status<class_ASAuthorizationAppleIDCredential_property_real_user_status>` property can help you determine if the user is likely a real person or a bot.
