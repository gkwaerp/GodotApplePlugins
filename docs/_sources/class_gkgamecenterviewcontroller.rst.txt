GKGameCenterViewController
==========================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

This class is used to present the different Game Center dashboards and the way to use it is by calling one of the different static show\_\* methods in this class in a fire-and-forget mode.
